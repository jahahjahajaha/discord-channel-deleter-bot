Clear Messages Command

/clear

The /clear command allows you to delete multiple messages at once from a Discord channel. You can specify how many messages to delete and apply various filters such as message type, author, and more.

Requirements

  • Manage Messages Permission: You must have the Manage Messages permission in the server.
  • Bot Permissions: The bot must have the Manage Messages permission.

How to Use

Step 1: Invoke the Command

Type /clear in the channel where you want to delete messages.

Step 2: Specify Options

The /clear command offers several options:

  • amount (Required): Number of messages to delete (1-100).
  • type (Optional): Filter by message type.
    • All Messages: Deletes all types of messages.
    • User Messages Only: Deletes only messages sent by regular users.
    • Bot Messages Only: Deletes only messages sent by bots.
    • Except System Messages: Deletes all messages except system messages.
  • from (Optional): Delete messages only from a specific user.
  • include_pinned (Optional): Whether to include pinned messages in the deletion.

Step 3: Execution

After entering your options, press Enter to execute the command. The bot will show a processing message.

Step 4: Completion

The bot will delete the messages according to your criteria and show a summary of the results.

Command Examples

Example 1: Delete 10 messages

/clear amount:10

This deletes the 10 most recent messages in the channel.

Example 2: Delete 50 bot messages

/clear amount:50 type:bot

This deletes up to 50 of the most recent messages sent by bots.

Example 3: Delete messages from a specific user

/clear amount:100 from:@username

This deletes up to 100 of the most recent messages sent by the specified user.

Example 4: Delete messages including pinned ones

/clear amount:20 include_pinned:true

This deletes the 20 most recent messages, including any that are pinned.

Common Issues

Discord API Limitations

Discord only allows bulk deletion of messages that are less than 14 days old. Older messages must be deleted individually, which is slower.

Permission Errors

If you or the bot don't have the Manage Messages permission, the command will fail. Check both your permissions and the bot's permissions.

Maximum Amount

Discord limits bulk deletion to 100 messages at a time. If you need to delete more, you'll need to run the command multiple times.
