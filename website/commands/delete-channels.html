Delete Channels Command

/delete-channels

The /delete-channels command allows you to delete multiple channels at once from your Discord server. You can select which channels to keep, and all other channels will be deleted. This is particularly useful for server cleanup or reorganization.

Requirements

  • Administrator Permissions: You must have the Administrator permission in the server.
  • Bot Permissions: The bot must have the Manage Channels permission.

How to Use

Step 1: Invoke the Command

Type /delete-channels in any channel of your Discord server.

Step 2: Choose Channel Type Filter

The bot will present you with a dropdown menu to filter channels by type. You can choose from:

  • All Channels: Show all channel types
  • Text Channels: Show only text channels
  • Voice Channels: Show only voice channels
  • Categories: Show only category channels
  • Announcement Channels: Show only announcement channels
  • Forum Channels: Show only forum channels

Add your own screenshot here showing the filter dropdown

Step 3: Select Channels to Keep

After selecting a filter, you'll see a dropdown menu of channels. Select all channels you want to KEEP. All other channels will be deleted.

Tip: For servers with many channels, the bot provides pagination. Use the "Next Page" and "Previous Page" buttons to navigate through the list.

Step 4: Review and Confirm

After selecting channels to keep, review your selection and click "Confirm Selection".

Warning: This action is irreversible! Double-check your selection before confirming.

Step 5: Final Confirmation

The bot will ask for a final confirmation. Click "Yes, Delete Channels" to proceed with the deletion.

Step 6: Completion

The bot will process the deletion and show a summary of the results, including how many channels were successfully deleted and any failures.

Common Issues

Permission Errors

If the bot doesn't have sufficient permissions, you'll see an error message. Make sure the bot has the "Manage Channels" permission.

Protected Channels

Some channels might be protected by Discord and cannot be deleted (e.g., Community servers' rules channels).

Command Timeout

The command has a timeout of 15 minutes. If you don't complete the selection within this time, you'll need to restart.

Related Commands

  • /delete-roles - Delete multiple roles at once
  • /clear - Delete multiple messages at once
