Delete Roles Command

/delete-roles

The /delete-roles command allows you to delete multiple roles at once from your Discord server. You can select which roles to keep, and all other roles will be deleted. This is particularly useful for cleaning up unused roles or reorganizing your permission system.

Requirements

  • Administrator Permissions: You must have the Administrator permission in the server.
  • Bot Permissions: The bot must have the Manage Roles permission.
  • Role Hierarchy: The bot's highest role must be above the roles you want to delete.

How to Use

Step 1: Invoke the Command

Type /delete-roles in any channel of your Discord server.

Step 2: Choose Role Filter

The bot will present you with a dropdown menu to filter roles by type. You can choose from:

  • All Roles: Show all roles
  • High Priority Roles: Show only high priority roles (position > 15)
  • Medium Priority Roles: Show only medium priority roles (position 5-15)
  • Low Priority Roles: Show only low priority roles (position < 5)
  • Displayed Separately: Show only roles displayed separately in the member list
  • Colored Roles: Show only roles with custom colors

Step 3: Select Roles to Keep

After selecting a filter, you'll see a dropdown menu of roles. Select all roles you want to KEEP. All other roles will be deleted.

Tip: For servers with many roles, the bot provides pagination. Use the "Next Page" and "Previous Page" buttons to navigate through the list.

Step 4: Review and Confirm

After selecting roles to keep, review your selection and click "Confirm Selection".

Warning: This action is irreversible! Double-check your selection before confirming.

Step 5: Final Confirmation

The bot will ask for a final confirmation. Click "Yes, Delete Roles" to proceed with the deletion.

Step 6: Completion

The bot will process the deletion and show a summary of the results, including how many roles were successfully deleted and any failures.

Common Issues

Permission Errors

If the bot doesn't have sufficient permissions, you'll see an error message. Make sure the bot has the "Manage Roles" permission and its highest role is above the roles you want to delete.

Managed Roles

Roles that are managed by integrations (such as bot roles or server boost roles) cannot be deleted. The bot will automatically skip these roles.

Empty Role Names

For roles with empty names, the bot will display "Role (ID: [role-id])" to help identify them.

Command Timeout

The command has a timeout of 15 minutes. If you don't complete the selection within this time, you'll need to restart.

Related Commands

  • /delete-channels - Delete multiple channels at once
  • /clear - Delete multiple messages at once
